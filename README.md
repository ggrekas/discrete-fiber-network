# Developed Software simulating collagen matrix deformations through a discrete model. 

Code developed and used for the simulations of the paper: Compressive Instabilities Enable Cell-Induced Extreme Densification Patterns in the Fibrous Extracellular Matrix: Discrete Model Predictions, [available on arxiv](https://arxiv.org/abs/2212.01213).



### Software requirements

* FEniCS up to version 2019.1.0 with mshr support, ([installation instructions](https://fenicsproject.org/download/)).
